# Customer Churn Analysis 📊

## 📌 Project Overview

This project analyses customer churn data to identify patterns affecting customer retention. The goal is to uncover key insights and provide actionable business recommendations.

---

## 🛠 Tools & Technologies

* Python (Pandas, NumPy)
* Jupyter Notebook
* Data Cleaning & EDA

---

## 📊 Key Insights

* Approximately **26.5%** of customers have churned
* **Month-to-month contracts** show the highest churn
* **Fiber optic users** have higher churn rates
* **Long-term contracts** significantly reduce churn

---

## 💡 Business Recommendations

* Encourage long-term contracts through offers and discounts
* Improve service quality and pricing for high-risk groups
* Run retention campaigns for at-risk customers
* Introduce loyalty programs to improve retention

---

## 📁 Dataset

Telco Customer Churn Dataset
